1. Linux Commands

  • pwd - Prints Current Working Directory
  • ls – Directory Listing
  • cd dir - Change Directory To dir
  • mkdir dir – create a Directory dir
  • clear - clears the current window
  • history - lists the complete history of commands used
  • exit - exits the current directory
  • rm filename - removes the required file
  • rm -rf foldername - removes the required folder
